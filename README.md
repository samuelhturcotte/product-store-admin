# Instructions

```
yarn
npx lerna bootstrap
yarn dto:compile
```

Then run using following commands:

In order to run frontend:

```
web:start
```

Go to [http://127.0.0.1:5173](http://127.0.0.1:5173)

In order to run server:

```
server:start
```

Server runs on http://127.0.0.1:3002 by default
